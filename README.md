# Users CRUD App 🚀

A simple **User Management CRUD Application** built using **Node.js, Express.js, MongoDB, and Mongoose**.  
This project allows users to be created, viewed, updated, and deleted using CRUD operations.

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS
- HTML & CSS

## ✨ Features

- Create users
- View users
- Update user details
- Delete users
- MongoDB database integration

## 📂 Project Setup

Clone the repository:

```bash
git clone https://github.com/Manasviii28/CRUDusersapp.git
```

Install dependencies:

```bash
npm install
```



Run the application:

```bash
npm start
```

Open:

```
http://localhost:3000
```
